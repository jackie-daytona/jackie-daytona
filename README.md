## Hello there 👋

I'm Danny -- software engineer with a background in airline operations and a 
focus on AI/data tooling. Currently working as an AI Strategy Co-Op at Delta Air Lines and starting an MS in Computer Science 
at Georgia Tech this Fall. I like building things that are useful.

#### What I'm working on...
* **kpai** -- a full-stack AI-powered KPI evaluation tool built with 
FastAPI, PostgreSQL, and the Anthropic Claude API. Station-level airline 
performance dashboards with AI-generated analysis.
* **Mistborn RPG** -- a text adventure built with ASP.NET Core and the 
Anthropic API, set in Brandon Sanderson's Mistborn universe. Side project 
doubling as a way to learn C#, .NET, and Angular.
* **Fantasy Golf League** -- upgrading my dad's golf league from pen and 
paper to a real web app using Java, Spring Boot, and Angular.
